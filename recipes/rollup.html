Keto Rollups

Description

This is a easy recipe for the busy days / Lunch

Ingredients

  • Cheese of choice(I prefer cheddar)
  • Italian dressing if wanted for extra zang
  • Salt and Pepper
  • Deli Meat

Steps

  1. Layer cheese, and meat together
  2. Add dressing if wanted
  3. Roll up cheese into circle roll
  4. serve
